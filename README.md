# Smart Shopping Trolley

## Overview

Smart Shopping Trolley is an Arduino-based retail automation project designed to simplify the shopping and billing process. The system uses RFID technology to automatically identify products, calculate the total bill in real time, and display purchase details on an LCD screen. This helps reduce long billing queues and improves the overall shopping experience.

## Features

* RFID-based product identification
* Real-time billing and price calculation
* LCD display for product and billing information
* Duplicate item detection
* GSM-based bill notification via SMS
* Wireless communication support using Bluetooth/Wi-Fi modules
* User-friendly and cost-effective design

## Components Used

* Arduino Uno/Nano
* RFID Reader (MFRC522)
* RFID Tags
* 16x2 LCD Display
* GSM Module
* Bluetooth/Wi-Fi Module (Optional)
* Buzzer
* Power Supply
* Breadboard and Jumper Wires

## Working Principle

1. Each product is assigned a unique RFID tag.
2. The RFID reader scans the product when placed inside the trolley.
3. Arduino processes the tag information and retrieves product details.
4. Product name and price are displayed on the LCD.
5. The total bill is updated automatically.
6. At checkout, the bill can be sent to the customer through GSM communication.
